<Sobre Mim/>

Apaixonado por games e tecnologia, logo cedo decidi me aprofundar na programação. Sou obcecado em aprender e curioso para saber como tudo realmente funciona, acretido que a curiosidade é fundamental como a dedicação para continuar aprendendo e foi isso que me motivou a chegar até aqui. Atualmente estou focado e apaixonado em criar automações úteis e criativas para o dia a dia com Python, mas também tenho profundo conhecimento web com PHP, o que similarmente, me fez desenvolver habilidades em HTML, CSS e JS.

<Projetos/>

Mini Blog

O projeto que eu mais aprendi ao criar ele! Um mini blog pessoal, onde tive que criar um mini System Design para implementar ele. Com funções de autenticação de usuário feita do zero, padrões de arquitetura de software (como Middlewares, Services, DTOs, etc), este blog permite que usuários comentem em publicações criadas pelo admin (que pode ser mudado nas configurações do arquivo .env) e comentem em outras respostas.

PHP, Laravel 10, Bootstrap v5

GitHub

Kanban

Este projeto é um Quadro Kanban (gerenciador de tarefas) simples, que permite os usuários organizarem suas tarefas em "To Do", "Doing" e "Done". Este projeto foi meu primeiro aprofundamento em autenticação de usuário, sendo feito pelas ferramentas do Jetstream.

PHP, Laravel 10, Bootstrap v5, Jetstream v4

GitHub

Cadastro de Produtos

Um dos meus primeiros projetos de automação. O programa consiste em ler uma tabela excel com vários produtos, e salvar cada um em um site. Uma tarefa que poderia ser repetitiva no trabalho, pode ser automatizada com algumas linhas de código.

Python, Selenium, Openpyxl

GitHub

Automação Climática

Uma automação criada a partir de uma API de clima. O projeto consiste em pegar dados meteorológicos da API (Weather API) e mandar um email para um destinatário específico. Combinando este programa com o Task Scheduler do Windows, você pode receber um email todos os dias com os dados daquele dia.

Python, Weather API, Requests, SMTP Lib

GitHub
